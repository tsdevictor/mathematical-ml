# MNIST Merge Notes

The original standalone `mnist` repository has been merged into `mathematical-ml`.

The full original repo is preserved here:

    archive/mnist-original/

The original source scripts from that repo are also promoted into the package here:

    src/mathematical_ml/mnist/
      feedforward.py
      train.py
      evaluate.py

This keeps the original implementation available while making the public GitHub structure easier to navigate.

The merged MNIST code remains close to the original project code. It has not been replaced by a rewritten neural-network implementation.
